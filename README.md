# 555 Timer Astable Multivibrator

## Overview
A 555 timer configured in astable mode designed using KiCad and verified using LTspice.

## Features
- LTspice simulation
- KiCad schematic
- PCB layout
- Gerber files
- Through-hole components

## Tools Used
- KiCad 9
- LTspice
